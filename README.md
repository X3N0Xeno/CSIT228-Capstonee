Project Description
> JavaBox is a local-multiplayer party game suite built in Java 21+ using JavaFX. It acts as a centralized game hub that contains a catalog of games for couch-play bringing competitive energy to the party without needing multiple setups. Bringing classic games with a modern twist. Put your game faces on and SPICE.UP.THAT.PARTY!!!!!

Features
> Centralized game hub with a tile-based game launcher
> Per-game background music with automatic crossfade transitions
> Randomized SFX pitch variation for satisfying, non-repetitive audio feedback
> Animated roulette wheel with dramatic 3-spin reveal for Hangman
> Persistent win/loss/streak statistics for Hangman
> Screen shake and glow effects in Knucklebones for game juice
> Fully custom on-screen keyboard for Hangman input
> Pop-up window spawning system with escalating difficulty in Window Swarm
> Clean quit-to-lobby callback system so all games return seamlessly to the hub

Technologies
> Java 21+ (compiled at source/target 25)
> JavaFX 21.0.6 — UI, FXML views, animations, media playback
> Maven — build and dependency management
> JUnit 5.12.1 — unit testing
> File I/O — flat-file persistence for statistics (BufferedReader / BufferedWriter)
